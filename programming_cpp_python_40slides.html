C++ — An Overview
⚡
C++
"Close to the metal"
Creator: Bjarne Stroustrup (1985)
Origin: Extension of C with OOP features
Known for: Performance, speed, and control
Best for: Games, systems, embedded devices
Python — An Overview
🐍
Python
"Batteries included"
Creator: Guido van Rossum (1991)
Design: Readability and simplicity first

Status: One of the most popular languages today

OOP in Python — Class Example
main.py 
class Car:
    def __init__(self, brand, year):
        self.brand = brand
        self.year = year

    def start(self):
        print(f"{self.brand} is starting!")

my_car = Car("Toyota", 2023)
my_car.start()
🐍 __init__ is the constructor — runs when object is created

Error Handling — C++
main.cpp 
try {
    int result = 10 / 0;
} catch (exception &e) {
    cout << "Error: " << e.what();
}
try

throw
⚡ Uses try, catch, and throw keywords
Error Handling — Python
main.py 
try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")
finally:
    print("This always runs.")
✨ Cleaner syntax, easier to read — finally always executes!
File Handling — C++

File Handling — Python
file.py 
with open("data.txt", "w") as file:
    file.write("Hello, Python!")
🎯 with automatically closes the file — no leaks!
